# Go-Resto

## Problem description

Create an application to manage menu in a restaurant. Features:
- Add menu
- See the list of menu
- Edit menu

## Dev environment setup

Prerequisites:
- Ruby 2.7.0

Install dependencies
```shell
$ yarn
$ bundle install
```

Migrate database
```shell
$ rake db:migrate
```
